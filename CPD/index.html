Introduction to HTML

What have I learnt?

How to implement tags and their uses for different styles, along with the importance of using HTML correctly when consdering accessibility for people with disabilties using a website. I learned about forking and the pros of doing this.

How did I use this in practice?

I used different tags in the body of a repository to style and list sentences. Used image & source tags.

What did I find hard?

I found the session easy to understand, my issue today was remembering to close tags correctly using the backslash. Other than this I found everything straightforward and I will ensure I check all closed tags thoroughly going forward.
